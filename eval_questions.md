# SwiftScale – 50 Eval Questions & Expected Answers

Use these to test your RAG system before deploying. Each question reflects a realistic user query.

## Pricing & Plans

1. How much does the Starter plan cost?
   → $29/month

2. What is the price of the Growth plan?
   → $99/month

3. How much is the Pro plan per month?
   → $299/month

4. Is Enterprise pricing fixed?
   → No, Enterprise is custom pricing — contact sales.

5. How much do I save with annual billing?
   → 2 months free (~17% off). Starter: $290/yr, Growth: $990/yr, Pro: $2,990/yr.

6. Is there a free plan?
   → Starter has a perpetual free tier with limited features.

7. Is there a free trial for Pro?
   → Yes, 14-day free trial, no credit card required.

8. Does Growth have a free trial?
   → Yes, 14-day free trial.

9. Does Starter have a free trial?
   → Starter has a free tier, not a time-limited trial.

10. What payment methods are accepted?
    → Credit cards (Visa, Mastercard, Amex), PayPal, and bank transfers for Enterprise.

## Users & Storage

11. How many users can I have on Starter?
    → Up to 3 users.

12. What is the user limit on the Growth plan?
    → Up to 20 users.

13. How many users does Pro support?
    → Up to 100 users.

14. Is there a user limit on Enterprise?
    → No, unlimited users.

15. How much storage does the Starter plan include?
    → 10 GB.

16. What is the storage limit on Growth?
    → 100 GB.

17. How much storage is included in Pro?
    → 500 GB.

18. Does Enterprise have a storage limit?
    → No, unlimited storage.

## API Calls

19. How many API calls does Starter include per month?
    → 5,000 API calls/month.

20. What is the API call limit on Growth?
    → 50,000 API calls/month.

21. How many API calls does Pro include?
    → 500,000 API calls/month.

22. What happens if I exceed the API limit on Starter?
    → Calls are hard-capped; you get a warning email at 80%.

23. What is the overage cost on Growth or Pro?
    → $0.002 per additional API call.

24. Does Enterprise have API overage charges?
    → No, Enterprise has unlimited API calls with no overage.

## Features

25. Which plans include SSO?
    → Pro (Google and Okta) and Enterprise (any SAML 2.0 provider).

26. Is SSO available on the Growth plan?
    → No, SSO is only on Pro and Enterprise.

27. How many workspaces does Growth allow?
    → 5 workspaces.

28. Can I create unlimited workspaces on Pro?
    → Yes.

29. Does Starter support custom domains?
    → No, custom domains are only on Pro and Enterprise.

30. Which plans include Salesforce integration?
    → Pro and Enterprise.

31. Does Growth include HubSpot integration?
    → No, HubSpot is on Pro and Enterprise only.

32. What integrations does Growth include?
    → Zapier and Slack.

33. Is there a mobile app?
    → Yes, iOS and Android on all paid plans.

34. Can I export data on the Growth plan?
    → Yes, CSV exports are available on Growth.

35. What export formats does Pro support?
    → CSV, JSON, and PDF.

36. Does SwiftScale have a REST API?
    → Yes, available on all plans. Full read/write on Pro and Enterprise; read-only analytics on Growth.

## Support

37. What is the support response time on Starter?
    → Email support with 48-hour response.

38. How fast is support on Growth?
    → Priority email with 24-hour response.

39. What support does Pro include?
    → Live chat with 4-hour response.

40. What support does Enterprise get?
    → Dedicated account manager with 1-hour SLA and custom SLA options.

41. Is there a knowledge base?
    → Yes, docs.swiftscale.io with video tutorials and a community forum on all plans.

## Security & Compliance

42. Is SwiftScale SOC 2 certified?
    → Yes, SOC 2 Type II certified.

43. Is SwiftScale GDPR compliant?
    → Yes, with a DPA available for all paid customers.

44. Where is data stored by default?
    → US-East on AWS. Pro and Enterprise can choose EU-West or Asia-Pacific.

45. Does SwiftScale offer on-premise deployment?
    → Yes, on the Enterprise plan only.

46. Are audit logs available?
    → Yes, on Enterprise only, with 12-month retention.

47. Is MFA supported?
    → Yes, on all plans.

48. What encryption does SwiftScale use?
    → AES-256 at rest, TLS 1.3 in transit.

## Billing Edge Cases

49. Can I get a refund?
    → Yes, 30-day money-back guarantee on first-time subscriptions (Starter, Growth, Pro). Enterprise is non-refundable.

50. Can I downgrade my plan?
    → Yes. Downgrades take effect at the start of the next billing cycle.
